Math can be rendered with Mathjax!
------------------------------
This code:
    Mathjax display
    \\[ {e}^{i\pi }+1=0 \\] 
    Mathjax inline \\( {e}^{i\pi }+1=0 \\)

produces
Mathjax display
\\[ {e}^{i\pi }+1=0 \\] 
Mathjax inline \\( {e}^{i\pi }+1=0 \\)

Let's try putting math in a list

1. Create a list
2. Stick an inline formula like \\(2+2\neq5\\) in the list
2. Stick a display formula like \\[2+2\neq5\\] in the list
3.  Now how about a nested list
    + Stick an inline formula like \\(2+2\neq5\\) in the list
    + Stick a display formula like \\[2+2\neq5\\] in the list

What about quotes
> My inline friend said  \\(2+2\neq5\\)
> My display friend said \\[2+2\neq5\\]
